EN
Project created for a college class where the objective is:
- Create a Java system with login and password plus the option to register using at least 3 user attributes.
- Users can register events that have occurred in their city.
- Users can search for events registered by other users and check in publicly or anonymously.
- Users can see who is attending the event by viewing public check-ins.
- Users can see which events they have checked in to and sort them.
- If a user cannot attend an event, they can cancel the check-in.
- The system can show the user events that have already taken place.
